# Sales Analysis Dashboard 📊

A Python-based sales data analysis project that explores monthly sales trends, category performance, and regional distribution.

## Features
- Monthly sales visualization
- Sales breakdown by category and region
- Summary statistics (best/worst month, totals, averages)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Results
- Total Annual Sales: $604,843
- Best Month: December ($77,678)
- Top Category: Beauty (49.6%)
- Top Region: West
